## Laravel ToDo-list
<p>使用Laradock 建置Laravel 8，資料庫:MySQL</p>

## Features
* 在Full calendar上讀取該月份資料
* 使用者可點選日期並新增、更新、刪除、查看資料
* 使用者可新增與刪除分類

- [x] Vue.JS
- [x] axios
- [x] Bootstrap Vue
- [x] PHPUnit
- [x] API 文件
